境外結構型商品專區

什麼是境外結構型商品？

Fixed Coupon Note (FCN)

提供投資人比傳統定存更高的固定利息，並以連結標的（如個股或ETF）的表現作為條件。特色在於定期支付利息，但不保本，若連結標的價格在到期時跌破履約價格，投資人可能損失部分或全部本金。

Stepdown FCN

一般FCN的延伸架構，提供同樣的固定收益機制，並設有逐期遞減的提前出場價格，每期最後一日觀察是否提前出場。適合預期市場震盪偏多、希望提高中後期出場機會、追求穩定配息且能承擔一定標的下跌風險的投資人。

Bullet FCN

商品利息只在第一個配息日配發，所以越早提前出場，換算的年化配息率越高，適合在多頭市場時選擇的架構。

Daily Range Accrual Note (DRAN)

浮動配息的結構型商品架構，提供投資人高於FCN的浮動配息收益。配息依標的股票價格表現浮動，若任一連結標的價格跌破履約價格 (Strike)，當天配息將不計，並有觸發自動提前出場機制 (KO) 即到期以履約價 (Strike) 轉換股票的風險。

Bond Linked Note (BLN)

連結債券的結構型商品，每個月固定配息，發行後觀察到期時債券的價格來判斷是否承接債券，適合尋求債券進場時點的投資人選擇。

Regular BEN

連結至少兩檔股票，到期時觀察表現最差者，股票價格高於履約價就可領取約定紅利率(Bonus)，漲幅超過其紅利率就能100%參與漲幅。若到期跌破履約價，投資人需以履約價格承接該股票，無紅利可得。適合能承擔股票風險，在市場處於多頭行情時的投資工具。

BEN with Lookback

一般BEN的延伸架構，以表現最差股票決定收益，但會採發行日起算N日的回顧期間最低價作為期初價格，能在無法掌握進場時機或市場較為波動時取得相對有利的進場點，適合希望提高進場彈性的投資人。

Digital Note (DCN)

與BEN相似，但連結標的可以為單一股票，到時期僅依標的價格是否達履約價判定能不能收到紅利率，不會參與任何上漲幅度，結構較為單純。

Booster

至少要連結兩檔股票，發行後直接觀察到期時最差股票表現。若最差股票跌破履約價格，就以履約價承接股票；若到期漲幅高於下檔觸發價(Lower Call Strike)，即可獲得漲幅乘以參與率的高額收益，適合看多市場的投資人。

保本Super BEN

具備每月觀察提前出場機制，保證領息期間後達到提前出場價格條件即可領回本金+年化出場紅利；如果沒發生提前出場事件，到期可以領出出場紅利並參與表現最差標的之漲跌幅，最高無上限，最低也不會損失到本金，結構上具有收益彈性與保本的特色，適合震盪偏多的市場及尋求穩健收益的投資人。

保本Callable BEN

為保本架構，發行機構擁有每月買回權，若被買回，投資人可領回100%本金+紅利率；若未被買回且到期時最差股票價格高於期初價，除領取本金+紅利外，還可以依參與率參與超過紅利率部分漲幅；若最終價格低於期初價格則僅返還投資本金，不會有額外損失。

保本KO Sharkfin

為每日觀察提前出場架構，如存續期間內股價超過提前出場價格即可立即出場，並領取出場紅利；若存續期間從未達到出場條件就持有至到期，到期跌多少拿多少，上漲就沒有收益。此商品為一個保本型看空架構，適合在空頭市場時選擇。

保本 Twinwin

設有上下界，存續期間內若股價從未突破到期時可以依實際漲跌參與收益，且能設定最低收益保障；如果曾經突破過上下界，則到期時不管股票表現如何，投資人收益皆是約定好的KO收益率。結構兼具保本、區間策略及彈性收益特性，適合在盤整震盪區間時選擇。

不保本 Twinwin

發行後每月觀察股價是否突破提前出場價，達成即可提前出場並領取年化配息；如未出場則觀察存續期間是否曾經觸及到下限價，曾觸及下限價者到期時需漲回履約價上才不用承接股票；若從未觸及，則到期時不論漲跌皆可以依漲跌幅獲利，具有雙向收益機會，適合看區間震盪即盤整行情的投資人。

Decumulator Note

為逐期用約定好高於市價的價格去賣出股票的結構型商品，可以在波動市場中提高平均賣出價格。適合希望分批賣出手上庫存標的，設定一個較高的價格且不想要一次性賣出的投資人。

Accumulator Note

為逐期用約定好低於市價的價格去買進股票的結構型商品，可以在波動市場中降低股票買進成本。適合希望分批買進股票，不想要一次性買入的投資人。

Steepener

觀察2個擔保隔夜融資固定期間交換利率(SOFR CMS)之間的利差，作為收益的計算基準，並可以設有最低及最高收益條件。利差越大期收益會越高，適合較為保守的投資人選擇。

Inverse Floater

設定一個利率基準扣除擔保隔夜融資固定期間交換利率(SOFR CMS)之間的利差，作為收益的計算基準，並可以設有最低及最高收益條件。掛勾的SOFR CMS越低，收益越高，適合預期降息的市場及較為保守的投資人選擇。

DRAPGN

設定一個利率區間作為收益的計算基準，如果掛鉤的利率價格跑出區間外，當天就不計息，適合較為保守的投資人選擇。
